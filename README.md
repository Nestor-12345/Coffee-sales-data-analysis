# Coffee Sales Analysis

## Project Overview

This project explores and analyzes a coffee sales dataset. The goal is to uncover patterns in sales trends, customer preferences, and product performance.
The analysis uses Python (Pandas, NumPy, Matplotlib, Seaborn) for data cleaning, exploration, and visualization.

## Dataset

Source: Coffee sales dataset (CSV format)

Content: Transaction records including product, quantity, date, and sales revenue

Size: ~3547 rows × 11 columns (fill in after checking your dataset)

## Tools & Libraries

Python

Pandas, NumPy → Data cleaning & analysis

Matplotlib, Seaborn → Data visualization

Jupyter Notebook / Kaggle Notebooks

## Key Insights

Best-selling coffee products by category

Seasonal or monthly sales trends

Revenue contributions by product type

Customer purchasing behavior
